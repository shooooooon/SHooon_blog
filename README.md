# Shooon Blog

モダンでシンプル、読みやすい個人ブログサイト

## 概要

2025年時点でモダンとされるUI/UXデザインを反映した個人ブログサイトです。テクノロジー、デザイン、ライフスタイルについて発信しています。

## 主な機能

- **レスポンシブデザイン**: モバイルファーストで全デバイスに最適化
- **ダークモード対応**: ライト/ダークテーマの切り替え
- **Stickyナビゲーション**: スクロールしても常にアクセス可能なヘッダー
- **記事管理**: カテゴリ・タグによる整理と検索機能
- **SEO対策**: メタタグ、OGP、構造化データ対応
- **高速表示**: LazyLoading、画像最適化

## 技術スタック

- **フロントエンド**: React 19 + TypeScript
- **スタイリング**: Tailwind CSS 4
- **UIコンポーネント**: shadcn/ui
- **ルーティング**: Wouter
- **ビルドツール**: Vite

## ページ構成

- **Home**: ヒーローセクション、注目記事、最新記事
- **Blog**: 記事一覧、検索・フィルタリング
- **BlogPost**: 記事詳細、関連記事、SNSシェア
- **Category/Tag**: カテゴリ・タグ別記事一覧
- **About**: プロフィール、SNSリンク
- **Privacy/Terms**: プライバシーポリシー、利用規約

## デザイン

- **カラー**: パープル系のアクセントカラー
- **フォント**: 
  - 本文: Inter
  - 見出し: Playfair Display
- **レイアウト**: ミニマルでクリーン、余白を広く取った可読性重視のデザイン

## 開発

```bash
# 依存関係のインストール
pnpm install

# 開発サーバーの起動
pnpm dev

# ビルド
pnpm build

# プレビュー
pnpm preview
```
